Project 1

Diptyque Holiday Web Redesign

국내 딥디크 웹사이트는 신세계몰 내 부속 페이지로 운영되어
브랜드의 감성과 철학을 전달하지 못하고, 다양한 기기에서의
사용자 경험도 미흡합니다. 또한 영어 기반의 공식 사이트는
국내 사용자의 이해와 구매에 불편함을 줍니다.
이에 따라, Holiday 시즌 컬렉션을 중심으로 브랜드 무드와
사용성을 반영한 반응형 웹사이트로 리디자인 했습니다.

  • 상단 GNB가 1Depth로만 구성되어 있어 제품 탐색 흐름 단절

UX PART

  • 디바이스별 최적화된 반응형 설계
  • 한국어 톤앤매너 반영으로 딥디크의 고급스러움, 예술성, 향의 분위기 전달
UI PART
  • 홀리데이 컬렉션에 맞춰 딥한 겨울톤 이미지와 톤 다운된 색상 팔레트 사용
  • 풀 스크린 이미지와 매거진형 레이아웃을 기반으로, 사용자의 시선 흐름에 맞춘 자연스러운 스크롤
  • Serif 중심으로 딥디크의 유럽 감성과 고급스러움 강조

Design Concept

Bohemian Luxe / Minimalism / Classic Holiday / Winter Nature

딥디크 특유의 앤티크한 감성과 예술적 자유로움을 바탕으로,
깊이 있는 명암과 정제된 오브제 연출, 유럽 전통의 따뜻한 연말 무드,
그리고 겨울의 고요한 분위기를 조화롭게 담아낸 홀리데이 시즌 디자인 컨셉입니다.
고급스럽고 차분한 구도 속에 브랜드 무드와 계절성을 세련되게 풀어냈습니다.

Design Guideline

텍스트보다는 이미지 중심의 정보 전달에 초점을 맞춰 제품을 직관적으로 나타내고자 했습니다.
제품의 특징이 시각적으로 먼저 전달되도록 구성하여, 사용자의 몰입도를 높이고 정보 흐름이 자연스럽도록 설계했습니다.

GNB

데스크탑에서는 상단 고정형 내비게이션 바를 통해 마우스 hover 시 드롭다운 메뉴가 활성화되고,
카테고리-서브 카테고리-컬렉션의 3단 구조로 사용자가 빠르게 원하는 정보를 탐색할 수 있도록 설계했습니다.
또한, 오른쪽 이미지 영역은 사용자의 시선을 유도하여 주력 제품 및 컬렉션을 자연스럽게 노출하도록 구성했습니다.
