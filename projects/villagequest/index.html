Project 2

Village Quest Website

메타버스 플랫폼 ZEP에서 제작한 픽셀 게임을 소개하고
플레이를 유도하며, 프로젝트 팀까지 함께 소개하는 컨셉 웹사이트입니다.
게임의 레트로 무드와 픽셀 그래픽을 웹에서도 이어갈 수 있도록
윈도우95 UI를 모티브로 하여 제작했으며, 사용자가
단순히 정보를 보는 것을 넘어 몰입과 인터랙션과
재미를 경험할 수 있도록 반응형 웹사이트로 제작했습니다.

Role

게임의 세계관 기획 과정은 팀 전체가 함께 진행했으며, 저는 이를 기반으로 웹사이트 기획과 웹디자인을 100% 전담하여 반응형 웹사이트로 제작했습니다.

  • Web Planning (100%)

    IA와 내비게이션 플로우를 기획했으며, 아이콘 클릭, 팝업, 스크롤 트리거 등 다양한 인터랙션을 반영하여 웹 탐색 과정을 게임 플레이처럼 체감할 수 있도록 구성했습니다.

  • Web Design (100%)

    Windows 95의 레트로한 감성을 바탕으로, 기존 아이콘과 비주얼 스타일을 현대적인 웹 환경에 맞춰 재구성하여 UI 컴포넌트를 디자인 하였으며, 최소 너비 500px까지 대응하는 반응형 웹으로 디자인 했습니다.

Design Concepts

Pixel Graphics
게임의 픽셀과 레트로 감성을 웹에서도
이어갈 수 있도록 Windows95 UI와 픽셀
그래픽 요소로 분위기를 반영하였습니다.
Playful Interaction
클릭, 이벤트, 스크롤 트리거를 통해
탐색 자체가 체험이 되도록
설계하였습니다.
One-page Flow
One-page IA와 앵커 스크롤로
사용자가 흐름을 끊지 않고 웹 전체에
몰입할 수 있도록 구성하였습니다.

Information Architecture

이 웹 사이트는 단일 페이지 IA로 설계되었으며, 섹션은 About → Review → Introduce → Ready 순으로 이어집니다.
헤더는 앵커 스크롤을 트리거하여 사용자가 원하는 섹션에 즉시 도달할 수 있도록 탐색 효율성을 높였습니다.

Design Guideline

Village Quest 웹사이트는 게임의 레트로 무드와 픽셀 그래픽을 웹에서도 이어가기 위해 원페이지 레이아웃으로 구성했습니다.
1700px 고정 폭과 여백 활용으로 스크롤 트리거와 가로 스크롤 등 인터랙션이 답답하지 않게 구현되도록 했으며,
반응형 설계는 최소 500px 너비를 기준으로 작업했습니다.

From Loading to Main Flow

사용자가 이어질 콘텐츠에 대한 기대감을 느낄 수 있도록,
게임을 실행하는 듯한 몰입감을 주는 로딩 화면과 사용자가 실제 화면 속으로 들어가는 전환을 설계했습니다.

About

메인 화면은 About 섹션 역할을 겸합니다. 아이콘 클릭 시 게임에 대한 콘텐츠를 확인하는 modal이 열리고,
아이콘은 자유롭게 드래그할 수 있어 실제 데스크톱을 조작하는 경험을 제공합니다.

Play CTA
우측 하단에 Play 버튼을 배치해
어떤 지점에서도 즉시 플레이로 전환할 수 있습니다.

Review

세로 스크롤 트리거로 리뷰를 한 줄씩 전개해 흐름을 끊지 않고 내용에 집중하도록 설계했습니다.

Our Team

팀 소개 섹션은 색상 반전과 가로 스크롤로 전 섹션과 뚜렷한 대비를 만들었습니다.
스크롤을 따라 스마일 이모지가 헤드라인을 가로지르며 생동감을 더해, 단순한 역할 소개를 상호작용 중심의 즐거운 경험으로 확장했습니다.

Respoinsive design

768px 이하 뷰포트에서는 가로 스크롤이 세로 스크롤로 전환되고,
팀 역할을 설명하는 두 개의 카드가 슬라이드 인터랙션으로 바뀌어 작은 화면에서도
가독성과 몰입감을 유지할 수 있도록 했습니다.

Ready

게임 링크로 이동할 수 있도록 윈도우 스타일 창 UI로 구성했으며,
Yes를 누르면 게임 링크로 바로 이동하고, No를 누르면 에러창이 출력되어, 단순한 선택을 재미있는 상호작용으로 연출했습니다.
